There are two sets of sources contained within these book sources.  The sources for running the demo web application reside within the JavaEERecipesSource directory.  The demo web application pertains to all of the recipes in the book, with the exception of those recipes in Chapter 18:  JavaFX in the Enterprise.  The sources for the JavaFX chapter reside within the JavaFX directory in the sources.

Further instructions for building the demo web application reside within the JavaEERecipesSource directory.
